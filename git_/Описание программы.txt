Приложение WPF для работы с БД. Используется подход Code First (Entity Framework), LINQ. Для работы с приложением необходимо восстановть БД из скрипта.
Программа реализует вывод всех товаров, сортировку по категориям (производители, категории, стоимость), добавление, удаление и редактирование товара.
Для корректного вывода данных о товарах описаны привязки и конверторы.
